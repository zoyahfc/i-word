🧠 Как работает нейронная сеть

Простое объяснение логики модели для Students Performance

📊 Сравнение с другими моделями

Модель Как работает Аналогия
Линейная регрессия y = w₁x₁ + w₂x₂ + b Прямая линия, проводящая связь между признаками и целью
Дерево решений Если признак > порог → влево/вправо Серия вопросов «да/нет», как в игре «20 вопросов»
Нейронная сеть Слой 1 → Слой 2 → Прогноз Команда экспертов, последовательно обрабатывающих информацию

🏗️ Архитектура вашей сети

Вход (4 признака) → Скрытый слой (2 нейрона) → Выход (1 прогноз)

📥 Шаг 1: Входные данные

После масштабирования каждый признак в диапазоне [0, 1]:

[gender=0, lunch=1, test_prep=1, reading=0.85]

🔗 Шаг 2: Скрытый слой (взвешенная сумма)

Каждый нейрон считает взвешенную сумму входов:

Нейрон А: (0×w₁) + (1×w₂) + (1×w₃) + (0.85×w₄) + bias → z₁

⚡ Шаг 3: Функция активации (сигмоида)

Сигмоида «сжимает» любое число в диапазон (0, 1):

σ(z) = 1 / (1 + e⁻ᶻ)

Это позволяет сети улавливать нелинейные зависимости.

📤 Шаг 4: Выходной слой

Выходной нейрон объединяет результаты скрытых нейронов и выдаёт прогноз math score.

🔁 Как сеть учится? (Backpropagation)

Аналогия: настройка старого радио 📻

Прямой проход: Вход → Сеть → Прогноз
Расчёт ошибки: loss = (прогноз - истинное)²
Обратный проход: Вычисляем, какие веса «виноваты» в ошибке (градиенты)
Обновление весов: weight = weight - lr × градиент
Повтор: 1000 эпох → ошибка уменьшается 📉

Эпоха 500 Ошибка: 0.012456 ⚠️
Эпоха 1000 Ошибка: 0.004016 🎯

✅ Почему ошибки такие низкие?

reading score Сильно коррелирует с math score
Масштабирование Все признаки в [0, 1] → стабильное обучение
Архитектура 4→2→1 Достаточно сложная, но без переобучения
1000 эпох + SGD Веса успели «устаканиться»

🔍 Что «видит» сеть в данных?

Линейная регрессия:

math = 0.65×reading + 0.12×lunch + 0.08×test_prep + 0.1

Дерево решений:

Если reading > 0.7:
  Если test_prep == 1 → math ≈ 0.85
  Иначе → math ≈ 0.72
Иначе → math ≈ 0.55

Нейронная сеть:

Сеть находит сложные комбинации признаков, которые человек мог бы упустить:

«Если чтение высокое И подготовка есть, но обед бесплатный →
математика будет средней, потому что...»

💡 Сеть не даёт явных правил, но автоматически учит нелинейные взаимодействия!

🧩 Ключевые выводы

1. Универсальный аппроксиматор Может выучить почти любую зависимость при достаточных данных
2. Backpropagation Сеть сама понимает, какие веса менять для уменьшения ошибки
3. Сигмоида = нелинейность Без неё сеть была бы просто линейной регрессией
4. MSE=0.004 Сеть хорошо уловила связь между признаками и целью
